# Astra M

`astra_m` — консольный Modbus TCP клиент на Rust для проверки связи с ПЛК/устройствами Modbus TCP.

Проект используется как диагностический инструмент для чтения регистров по Modbus TCP без HMI и без сторонних Modbus-библиотек.

## Назначение

Программа позволяет:

* подключаться к Modbus TCP устройству;
* задавать IP-адрес, порт, Slave ID, функцию чтения, стартовый адрес и количество регистров;
* циклически читать регистры;
* проверять поведение устройства и сети при длительном опросе;
* диагностировать зависания при чтении Modbus TCP.

## Поддерживаемые функции

* `03` — Read Holding Registers
* `04` — Read Input Registers

## Сборка

```bash
cargo build
```

## Запуск

```bash
cargo run
```

## Пример параметров

```text
IP: 192.168.0.10
Port: 502
Slave ID: 255
Function code: 4
Start address: 8
Register count: 2
Poll interval: 500 ms
```

## Примечание

Проект создан для диагностики связи с OWEN/ОВЕН ПЛК и проверки стабильности Modbus TCP соединения на Windows и Astra Linux.

Особое внимание уделяется ситуации, когда на Windows опрос работает стабильно, а на Astra Linux ARM64 чтение может зависать на ожидании ответа TCP/Modbus.
